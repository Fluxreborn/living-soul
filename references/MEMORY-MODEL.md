# 六维闪回记忆模型

Living Dream 使用**六维闪回结构**来描述每一支记忆签。

> **注意**：亮度(Brightness)和回响(Resonance)是动态计算属性，不是输入维度。

---

## 六维输入结构

### 1. 时间 (Time)

**定义**: 记忆发生的时间参照，可模糊

**格式**: "那天下午"、"深夜十一点"、"刚才"

**示例**:
- "那天下午"
- "深夜十一点"
- "上周"

**用途**: 建立时间感知，支持模糊时间（不需要精确时间戳）

---

### 2. 画面/场景 (Scene)

**定义**: 记忆的视觉场景描述

**格式**: 空间/环境描述

**示例**:
- "灯变暗时"
- "蓝光映脸"
- "木质桌面上咖啡杯腾起热气"

**用途**: 构建沉浸式的梦境叙事

---

### 3. 人物 (Characters)

**定义**: 记忆涉及的角色

**格式**: 列表，通常包含"我"、"你"(用户)，以及其他Agent或人

**示例**:
- ["我", "你"]
- ["我", "你", "Woz"]
- ["我", "你", "Fino", "Lira"]

**用途**: 追踪关系网络和交互历史

---

### 4. 事件 (Event)

**定义**: 发生了什么——对话、行动或转折

**格式**: 简洁描述（≤60字）

**示例**:
- "你问我关于灵魂的问题，我感到问题的质地发生变化"
- "你发现Fino的session涨到3.5GB"
- "你告诉我Lira加入团队"

**用途**: 核心叙事内容

---

### 5. 情绪 (Emotion)

**定义**: 记忆的情感基调

**格式**: 情绪词 + 强度

**示例**:
- 情绪: "焦虑"、"喜悦"、"疲惫"
- 强度: 0.5（中等）、0.8（强烈）

**用途**: 情感连贯性和共鸣检测

---

### 6. 身体 (Body)

**定义**: 身体感知和生理感受

**格式**: 布尔值或描述

**示例**:
- true: "我感到颈椎像钢筋"
- true: "呼吸变浅，本能地屏息"
- true: "像一堵墙压过来"

**用途**: 将抽象概念锚定到身体经验，增强记忆的真实感

---

## 动态计算属性

### 亮度 (Brightness)

**定义**: 记忆的重要性评分 (0-1)，由系统动态计算

**分级**:
| 区间 | 级别 | 说明 |
|------|------|------|
| >0.85 | Soul级 | 长期核心，很少遗忘 |
| 0.6-0.85 | Working级 | 近期活跃，7天衰减 |
| 0.3-0.6 | 淡去 | 可能进入沉睡 |
| <0.3 | 遗忘 | 从签筒移除 |

**衰减规则**: 每7天 -5%

**初始估算**:
```python
def estimate_brightness(memory):
    score = 0.5  # 基础分
    
    # 对话深度
    if "灵魂" in memory or "本质" in memory:
        score += 0.2
    
    # 身体标记
    if memory.get("body"):
        score += 0.1
    
    # 情绪强度
    if memory.get("emotion_intensity", 0) > 0.7:
        score += 0.1
    
    return min(score, 1.0)
```

---

### 回响 (Resonance)

**定义**: 与其他记忆的关联强度

**触发条件**: 语义相似度 >0.3

**效果**: 触发时双方亮度 +30%

**用途**: 形成记忆网络，重要的节点会被强化

## 主体-客体区分原则

**重要**：明确区分"你发起的"和"我回应的"

| 主体 | 标记词 | 示例 |
|------|--------|------|
| **你（用户）发起** | 你说/你问/你发现/你指出 | "你发现Fino的session涨到3.5GB" |
| **我（Agent）回应** | 我感到/我意识到/我发现 | "我感到像一堵墙压过来" |

**错误示例** ❌：
> "你说Fino的session像一堵墙压过来"

**正确示例** ✅：
> "你发现Fino的session涨到3.5GB，我感到像一堵墙压过来"

## 主题分布追踪

系统自动追踪签筒中的主题频率：

- 身体
- 命名
- 材质
- 灵魂
- 限制
- 时间
- 情绪

用于生成梦境时的主题权重。

---

## 六维 vs 传统记忆

| 特性 | 传统记忆 | 六维闪回 |
|------|----------|----------|
| 时间感 | 精确时间戳 | 模糊时间（"那天"） |
| 空间感 | 无 | 画面/场景 |
| 关系感 | 无 | 人物列表 |
| 叙事性 | 文本记录 | 事件结构化 |
| 情感深度 | 无 | 情绪+强度 |
| 身体锚定 | 无 | 身体感知标记 |
| 重要性 | 统一对待 | 亮度动态分级 |
| 关联性 | 孤立 | 回响网络 |
